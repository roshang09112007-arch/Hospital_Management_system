📖 About the Project

The **Hospital Management System** is a web-based application developed using HTML, CSS, and JavaScript. It helps in managing basic hospital functionalities such as doctor information, appointment booking, and user authentication.

This project focuses on providing a simple and interactive interface for users to access hospital services easily.



🔗 Live Demo

👉 https://amrita-hospital-management.netlify.app/



✨ Features

- 🏠 Home page with hospital introduction  
- ℹ️ About Us page describing hospital services  
- 👨‍⚕️ Doctors page with profiles and specializations  
- 📅 Appointment booking with date & time selection  
- 📞 Contact form for user queries  
- 🔐 User Registration system  
- 🔑 Login system  
- 📱 Responsive design
